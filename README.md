# AI-Game-Jam

- Create your branch
```
git checkout your-name
```

### Pushing to remote repository
- Fetch from remote origin/main first
```
git checkout main
git fetch
git log --all
git rebase origin/main
```
- Attempt to merge into main +/- fix merge conflicts
```
git merge your-name
```
- Push to remote
```
git push
```
